# 目录结构

```
/                       - 根目录
    src/                    - 开发目录
        api/                    - api接口
        assets/                 - 资源文件目录
            images/                 - 图片文件目录
        components/             - 组件库
        enum/                   - 枚举字段
        pages/                  - 路由页面
        styles/                 - 通用基本样式目录
        utils/                  - 通用方法
        App.vue
        main.js
        router.js
    dist/                   - 最终目录
    mock/                   - mock数据
```

# 项目运行

```
npm install

npm run dev

npm run build

```
